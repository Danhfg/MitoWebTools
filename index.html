Available Tools

RSCU Charts

Generate multiple interactive subplots of Relative Synonymous Codon Usage from various TSV files quickly, using Plotly.

Codon Usage

Analyze Codon Usage, Codon Usage per Thousand, or RSCU from FASTA or pre-processed files with interactive stacked bar charts.

Gene Synteny

Visualize gene synteny maps generated from BED annotations, with automatic offset for (+) and (-) strands and proper tRNA coloring.

D-Loop Analysis

Perfectly model flanking blocks and clustered Tandem Repeats circles from repetitive regions provided via text.

Fast in Browser

No backend servers, delays, or long queues. All parsing, processing, and graphic rendering occur client-side (Front-end).
